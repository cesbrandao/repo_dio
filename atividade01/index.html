Estrutura básica do HTML

HTML (Linguagem de Marcação de HiperTexto, em português) é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML possuem extensão .html e podem ser interpretados automaticamente por navegadores.

Um documento HTML é dividido em duas estruturas básicas: o HEAD, local no qual são especificadas informações que não serão exatamente visíveis ao usuário, como dados de uso e controle do documento; e BODY, estrutura na qual é estruturada a informação legível para o usuário do documento, ou seja, o documento em si.

Falando sobre Tags

Tags é o mecanismo delimitado pelos símbolos < > que é utilizado para estruturar um documento HTML. Alguns exemplos são:

  • h1 a h6, como tags de título e subtítulos.
  • p, como tag para delimitar o texto de um parágrafo.
  • ol e ul, para a criação de listas ordenadas ou não ordenadas, respectivamente.
  • li, utilizado para especificar cada item das listas citadas anteriormente.

Atributos das Tags

Atributos são informações que passadas dentro de uma tag para que ela se comporte da maneira esperada. Existem tanto atributos que funcionam em todas as tags (como atributo de identificação, id) quanto os específicos (como o atributo target da tag de hyperlink) que funcionam apenas em uma ou poucas tags.

Textos

As tags de texto definem textos, estilos de fonte, parágrafos, quebras de linhas, entre outras estruturas. Dentre as tags de texto, podemos citar a principal tag de texto, p, que delimita um parágrafo, a tag br, utilizada para quebra de linha ou a tag i, utilizada para estilizar o texto em formato itálico, por exemplo.

Listas ordenadas e desordenadas

Como apresentado anteriormente, em um documento HTML é possível a criação de listas, tanto uma lista ordenada, a partir da tag ol, quanto uma lista não ordenada, a partir da tag ul. Por fim, para a inclusão dentro da lista os seus elementos, utiliza-se a tag li.

Links

A tag de hyperlink é responsável por fazer a ligação entre páginas diferentes (link entre documentos). A tag utilizada para especificar tal estrutura é a tag a com o atributo obrigatório href="", com o diretório do documento ou link de outro domínio especificado entre aspas.
